# Design: Inner Activity Minimum Density (C23)

**Issue**: #176
**Milestone**: M12

## Problem Analysis

当前去 AI 化体系在"情感表达"维度上只防了一端：
- SP-07 禁止情绪标签句（"感到愤怒""心中涌起悲伤"）
- SP-01 禁止解释型旁白（"她知道他是在用这种方式表达关心"）

但缺少对另一端的约束——**完全不写内心活动**。AI 被训练避免情绪直述后，容易彻底跳过情感层，产出纯动作记录流。这种"情感扁平化"同样是明显的 AI 写作特征。

## Design Decisions

### D1: 约束定位——ChapterWriter C23

新增 `C23 内心活动锚点`，与 SP-07 形成互补闭环：

```
SP-07 防止 → "感到愤怒"（情绪标签）
C23 防止 → 连续纯动作流无内心活动（情感扁平）
合法表达 → 感官侵入 / 碎片思绪 / 生理反应 / 自我纠正
```

### D2: 触发条件——关键节点 + 连续动作流

两个触发维度：
1. **关键决策节点**：生死抉择、重大信息获知、SP 扣除等高压场景，必须在节点前后 2-3 句内有内心活动
2. **连续动作流上限**：纯动作记录（无内心活动、无感官描写、无碎片思绪）连续 ≤5 句，之后必须穿插角色感知

### D3: 合法内心活动类型

不是恢复"情绪标签"，而是用已有技法工具箱中的手段：
- `sensory_intrusion`（感官侵入）：铁锈味、皮肤发紧
- `stream_of_consciousness`（碎片思绪）：灯没关、药没吃
- `thought_interrupt`（思维中断）：想到一半被打断
- `self_correction`（自我纠正）：先说一个版本再改口
- 生理反应：手心出汗、喉咙发紧、呼吸变浅

### D4: QualityJudge 检测口径

在 `emotional_impact` 维度增加扣分规则：
- 关键决策节点前后 3 句无内心活动 → 扣 0.5 分
- 全章纯动作流超过 3 处连续 5 句 → 扣 1 分
- 不新增独立指标（避免 14th indicator 膨胀），归入 emotional_impact

### D5: StyleRefiner 检测

Step 4 节奏朗读测试中增加"纯动作流超长检测"：
- 扫描连续 5+ 句无内心/感官/思绪的纯动作段落
- 在动作流中间插入最小必要的感知片段

## Alternatives Considered

1. **新增第 14 个 indicator `inner_activity_density`**：被否决——指标体系已从 7→13 扩展过一次，再加会增加 QJ 认知负担。归入 emotional_impact 更自然
2. **固定配额（每千字 ≥N 处内心活动）**：被否决——与技法工具箱"不固定数量"的设计哲学冲突。用触发条件（关键节点 + 连续流上限）比固定配额更符合人类写作规律
3. **只在 StyleRefiner 后处理**：被否决——内心活动需要与情节深度绑定，后处理补入容易破坏语义。必须在 ChapterWriter 生成阶段就约束
